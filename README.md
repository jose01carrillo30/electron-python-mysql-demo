## Dependenices
  * Python
  * NodeJS
  * electron.js
  * python-shell 
    
    ## Usage
    * Make sure you have `electron.js` - download it by typing `npm install electron -g`. This is will install Electron.js and set up your PATH.
    * You can now run `npm start` inside the `gui/` directory.
